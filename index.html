Recipes

A recipe = a prompt body + which fields get appended under it + where the output is saved. Include an output field from an upstream recipe to chain steps.

Pipelines

A pipeline is an ordered list of recipes. Pick a subject once on the Generate tab, then step through: copy each prompt, paste its output, Save & next. Each step's output is saved before the next, so later steps automatically include earlier ones.
